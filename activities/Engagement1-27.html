Explain the difference between an absolute link and a relative link. Why would you use relative links?

An absolute link always starts with "https" and lead to other domains. Relative links lead within your own domain. The link should always start with the same domain name as the one you're using. Relative links are quicker to type and should make making our website faster when we use them instead of absolute links.

List the 3 rules with relative links.

  1. Use the file name
  2. If you're linking to another folder inside the folder you're using inside your folder, use "/" to navigate between folders. Use ".." to go "up" a folder.
  3. If you want to navigate direcly to the root of the site, start your link with a "/"
