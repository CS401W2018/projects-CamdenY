Engagement - Beginning CSS

Explain the difference between inline, embedded, and external CSS.

Internal CSS changes the style of one tag and uses "style =". Embedded CSS changes the style of all of one certain tag in the file using the "<style>" tag. It is put in the "head" section of the file. External CSS changes the style of your local file by taking the style from another css file using "link.css"

Explain why you should only use external CSS.

External CSS lets you create one style that can be used across many different files.

What is a selector used for?

A selector is the type of tag you want to change. You use the selector to specify what you want to stylize
