Engagement - CSS Selectors

List out the different ways to select an item.

  1. Tag the name of the tag
  2. Descendant Rule: Choose a tag that includes the tag you want to change in it.
  3. ID: Use an id before your text to attribute that tag to a variable associated with a style in your CSS file.
  4. Class: Similar to an ID but can be used multiple times. Multiple can be used on one tag.

What is the difference between an ID and a Class?

A Class can be used multiple times while and ID cannot. Multiple Classes can also be used on a single tag unlike ID's.

What is the difference between a SPAN tag and a DIV tag?

A div tag causes line breaks while span doesn't.

What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

Google understands that header, main and footer are different sections throughout the doc.

What are the different ways to select a color?

  1. Select by name
  2. RGB values
  3. Hex Code
  4. Short Code
  5. hSL
