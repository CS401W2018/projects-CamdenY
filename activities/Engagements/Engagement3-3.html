Engagement - Page for Print

Why is the viewport meta tag important?

Viewport ensures that the page works and looks correctly across all types of devices.

How can you adjust the CSS for printing?

Add the media tag and change it to "print" to link in order to adjust the page for printing.

When would you want to adjust the CSS for printing?

If we want our page to be printed, we should adjust it for printing.
