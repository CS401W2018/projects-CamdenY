Internet

A global network that connects devices together.

Web

A collection of public pages and sites that can found through the Internet.

How are the Internet and Web different?

The Internet is the network that the Web runs on. The Internet is like a road, while the Web is the destinations that can be found by walking on the road.

ASCII and Unicode

Two encoding schemes that encode characters into bits for storage and transmission.

What is UTF-8?

UTF-8 (or Unicode Transformation Format-8-bit) is a character encoding standard that uses 1-4 bytes to represent each character instead of ASCII's 1-8 bits.

URL

Stands for Uniform Resource Locator, a string of text that specifies a location on the Internet.

Domain Name

The string of text that specifies a certain website.

HTTP

Stands for Hypertext Transfer Protocol, a network protocol that transfers information between networked devices.

HTTPS

Stands for Hypertext Transfer Protocol Secure, a secure version of HTTP, meaning it encrypts communication between the user's device and the website.

IP Address

Unique number each device has that identifies it as it connects to the Internet.

DNS

Stands for Domain Name System, a naming service that identifies networks.

W3C Recommendations

A report that sets the standards and guidelines of the World Wide Web.

XML, XHTML, HTML, HTML5

Programming languages that are used to format webpages.
